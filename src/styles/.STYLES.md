# Styles Rulebook

- `globals.css` defines tokens. No imports.
- `themes.css` imports `./globals.css` and is the only place where themes live.
- `styles/<group>/.shared.css` imports exactly `../themes.css` (level 1) or `../.shared.css` (deeper levels).
- Every other CSS file inside that folder imports exactly `./.shared.css`.
- Apply the same rule recursively for any nested folder depth.
- No file skips a level or imports siblings.
- `styles/components/ui/.figma-ui-components.css` is exempt.
