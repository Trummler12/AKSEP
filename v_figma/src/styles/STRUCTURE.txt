Variable System
===============

Colors:
  --color-true-lavender         #7E65B9  (Primary dark)
  --color-horizon               #5A8FA8  (Middle)
  --color-mint                  #3EB694  (Primary light)
  --color-bg-darker             #322A45  (Background darker)
  --color-bg-darkerst           #423A51  (Background darkest)

Text Sizes:
  --text-xs → --text-6xl        (0.75rem → 3.75rem)

Spacing:
  --spacing-xs → --spacing-3xl  (0.25rem → 4rem)

Border Radius:
  --radius-sm → --radius-full   (0.375rem → 9999px)

Transitions:
  --transition-fast             150ms ease
  --transition-base             200ms ease
  --transition-slow             300ms ease

Z-Index:
  --z-base → --z-tooltip        (1 → 50)

Quick Reference
===============

Container Classes:
  .component-container         Standard container (1280px max)
  .page-container              Page container with padding
  .nav-header-container        Navigation container

Grid Layouts:
  .footer-grid                 1 → 3 columns (responsive)
  .hero-features-grid          1 → 3 columns
  .aktuelles-grid              1 → 2 → 3 columns
  .programm-grid               1 → 2 → 4 columns

Common Patterns:
  .nav-logo-link               Flex with gap, hover effect
  .nav-dropdown-trigger        Link with hover color change
  .nav-dropdown-item           Dropdown item with hover
  .hero-feature-icon-wrapper   Icon container with background

Icon Sizes:
  .*-icon                      Usually 1rem × 1rem
  .*-cta-icon                  Usually 1.25rem × 1.25rem
  .*-mobile-icon               Usually 1.5rem × 1.5rem
  .*-feature-icon              Usually 2rem × 2rem
